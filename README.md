# Azure Resource Group Deployment and Documentation Project

## Project Summary

This project demonstrates how to create, verify, document, and clean up a Microsoft Azure Resource Group. The purpose of this project is to show basic cloud administration skills that an IT Support or Help Desk professional may use when working with cloud-based environments.

A resource group is used in Azure to organize related cloud resources such as virtual machines, storage accounts, networks, and applications. This project shows the process of creating a resource group, confirming it was successfully deployed, and documenting the steps taken.
